# EEG-FPS-Game

The game is currently only compatible with Windows.

To play the game:
1. Install BlueMuse. https://github.com/kowalej/BlueMuse
2. Connect your Muse to your PC using BlueMuse. The game will not work unless your Muse is connected and streaming before starting the game.
3. Click on the "EEG FPS.exe" file to start the game.
